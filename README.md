# Dotfiles :: configs, scripts, prefs

Self described directories and files.

## Install

Checkout to ~/Projects folder (not a strict requirement, but recommended)
```
mkdir ~/Projects
git clone https://github.com/madeline-everett/dotfiles.git ~/Projects/dotfiles
bash ./action.install-to-local
```
